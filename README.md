# 👵 Connecta Senior

O **Connecta Senior** é uma plataforma desenvolvida para facilitar a socialização e o bem-estar da terceira idade, conectando idosos a eventos, atividades e recursos de saúde em suas comunidades. O projeto utiliza IA para oferecer recomendações personalizadas.

> 🎓 *Projeto desenvolvido para a matéria de Projeto Interdisciplinar do Instituto Federal do Paraná (IFPR) - Campus Pinhais.*

---

## 🚀 O que já foi feito

O projeto já possui sua estrutura base funcional e várias telas implementadas:

### 🛠️ Tecnologias Utilizadas
*   **React + TypeScript**: Interface moderna e tipada.
*   **Capacitor (Android)**: Transformação da aplicação web em um app nativo para celular.
*   **Vite**: Ferramenta de build rápida para o desenvolvimento.
*   **Google Gemini API**: Inteligência Artificial para sugestão de atividades.

### ✨ Funcionalidades Implementadas
*   **Interface Base**: Telas de Home, Atividades, Saúde, Chat e Perfil.
*   **Sistema de Notificações**: Gerenciador local de lembretes.
*   **Integração com IA**: Serviço configurado para utilizar o Google Gemini para recomendações.
*   **Geolocalização**: Estrutura preparada para exibição de mapas e rotas de atividades.
*   **Capacitor configurado**: Projeto pronto para gerar build Android (pasta `/android` disponível).

## 📋 O que está pendente (Roadmap)

Ainda há funcionalidades importantes para serem finalizadas ou refinadas:

*   **Autenticação**: Finalizar a integração real da tela de Login (Auth.tsx) com um banco de dados.
*   **Persistência de Dados**: Implementar um backend ou serviço de nuvem (como Firebase ou Supabase) para salvar o progresso dos usuários.
*   **Finalização do Chat**: Tornar o chat funcional para interação entre usuários ou com a IA.
*   **Refinamento de UX**: Ajustar a acessibilidade (tamanho de fontes e botões) focando especificamente no público idoso.
*   **Build Final**: Gerar e testar o APK final para dispositivos Android físicos.
